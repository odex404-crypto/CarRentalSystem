# CarRentalSystem
A **Java-based console Car Rental System** built using **OOP concepts**. It allows users to rent and return cars through a menu-driven interface. The system manages cars, customers, and rentals, checks availability, and calculates rental costs based on rental days using classes like Car, Customer, Rental, and CarRentalSystem.

# 🚗 Car Rental System (Java)

A simple console-based **Car Rental System** built using **Java and Object-Oriented Programming (OOP)**. The application allows users to rent cars, return cars, and manage rental records through a menu-driven interface while maintaining car availability and rental details.

## 📌 Features
- Rent available cars  
- Return rented cars  
- View available vehicles  
- Calculate rental cost based on number of days  
- Generate rental confirmation details  
- Menu-driven console interface  

## 🧠 OOP Concepts Used
- **Encapsulation** – Private variables with getter methods  
- **Classes & Objects** – Car, Customer, Rental, CarRentalSystem  
- **Object Interaction** – System manages relationships between cars and customers  
- **Collections** – Uses `ArrayList` to store cars, customers, and rentals
